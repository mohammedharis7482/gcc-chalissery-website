# GCC Chalissery Football Academy

## Website Design System v1.0

**Project:** GCC Chalissery Football Academy Official Website

**Document:** Design System

**Version:** 1.0

**Status:** Finalized (Planning Phase)

**Last Updated:** June 2026

---

# 1. Purpose

This document defines the official design system for the GCC Chalissery Football Academy website.

It serves as the single source of truth for every design and development decision throughout the project.

Every page, section, component, animation, and future feature must follow this design system to maintain consistency across the website.

This document should be referenced before making any design or development changes.

---

# 2. Brand Identity

## Brand Position

GCC Chalissery Football Academy is positioned as a premium football academy rooted in football legacy, discipline, youth development, and community values.

The website should communicate that GCC is not simply a local football club, but a professionally managed football academy that provides a structured pathway for young footballers.

## Brand Personality

The website should always feel:

* Premium
* Professional
* Modern
* Trustworthy
* Community Driven
* Football Focused
* Inspiring
* Confident
* Emotional
* Clean

The website should never feel like:

* A school website
* A sports blog
* A government website
* A dashboard
* A generic sports template
* A content-heavy document

---

# 3. Visual Design Philosophy

The existing website already establishes the desired visual identity.

This project focuses on improving content, structure, and user experience while preserving the current visual language.

The design direction should remain:

* Premium
* Dark
* Editorial
* Image-first
* Spacious
* Story-driven
* Modern
* Football-centric

Every section should contain one strong visual focal point supported by concise content.

Real academy photography should always be prioritized over decorative graphics.

---

# 4. Color System

The website will continue using the existing GCC color palette.

## Primary Colors

Deep Navy

Primary institutional color.

Used for:

* Hero sections
* Footer
* Final CTA
* Important feature sections
* Dark backgrounds

---

Royal Blue

Main brand color.

Used for:

* Feature sections
* Navigation accents
* Brand identity
* Cards
* Interactive elements

---

Accent Yellow

Used carefully to create emphasis.

Use only for:

* Primary buttons
* Active navigation
* AIFF badges
* Statistics
* Icons
* Small highlights
* Divider accents

Never use yellow as a large background color.

---

Light Blue

Used only as a secondary accent.

Used for:

* Hover effects
* Subtle gradients
* Motion accents
* Decorative highlights

It should never dominate the interface.

---

White

Used for:

* Light content sections
* Cards
* Typography on dark backgrounds
* Clean spacing

---

Off White

Used as the primary light surface throughout the website.

This provides visual breathing space between dark sections.

---

## Color Distribution

Instead of making every section dark, the website should alternate between dark and light sections.

Example page rhythm:

Dark Hero

↓

Light Story Section

↓

Royal Blue Feature Section

↓

White Content Section

↓

Dark Achievement Section

↓

Light Gallery

↓

Dark CTA

↓

Footer

This creates better readability while maintaining the premium football identity.

---

# 5. Typography

The website will continue using the existing typography system.

Primary Font

Geist Sans

Typography Style

* Large editorial headings
* Bold titles
* Comfortable reading experience
* Clear hierarchy
* Minimal text density

Guidelines

* Headings should be short and impactful.
* Paragraphs should remain concise.
* Avoid long document-like blocks of text.
* Use whitespace generously.

---

# 6. Layout System

The existing layout philosophy will remain unchanged.

Design Principles

* Editorial layouts
* Large imagery
* Split content sections
* Premium spacing
* Strong visual hierarchy
* One focal point per section

Avoid:

* Repetitive equal-sized grids
* Dense content blocks
* Dashboard-style layouts
* Overcrowded sections

---

# 7. Section Structure

Every page should follow a consistent structure:

Page Hero

↓

Main Story / Primary Content

↓

Supporting Content

↓

Proof / Statistics

↓

Gallery or Visual Section

↓

Call To Action

↓

Footer

Each section must have a clear purpose.

---

# 8. Component Library

The website should be built using reusable components.

Core Components

* Navbar
* Hero Section
* Section Header
* Primary Button
* Secondary Button
* Statistic Card
* Team Member Card
* Coach Card
* Achievement Card
* Player Card
* Gallery Card
* CTA Banner
* Footer

New components should only be introduced if absolutely necessary.

---

# 9. Card Design

Cards should feel premium and lightweight.

Characteristics

* Rounded corners
* Soft shadows
* Subtle glass effect where appropriate
* Clean spacing
* Strong image hierarchy
* Concise text
* Gentle hover lift

Cards should never resemble dashboard widgets.

---

# 10. Image Guidelines

Photography is one of GCC's strongest assets.

The website should primarily use real academy photography.

Priority order:

1. Training Sessions
2. Match Action
3. Team Photos
4. Coach & Player Interaction
5. Trophy Moments
6. Community Events
7. Registration Activities

Avoid stock football photography unless absolutely necessary.

Images should:

* Maintain natural colours
* Have proper cropping
* Preserve faces
* Use subtle overlays when text is placed over them

---

# 11. Motion & Animation

Animation should enhance the experience without becoming distracting.

Primary Motion Types

* Fade Up
* Fade In
* Card Lift
* Image Zoom
* Button Hover
* Section Reveal

Animation Principles

* Smooth
* Fast
* Premium
* Consistent
* Purposeful

Avoid excessive animations or effects that compete with the content.

---

# 12. Responsive Design

The mobile experience should receive equal attention.

Requirements

* Compact navbar
* Comfortable touch targets
* Proper spacing
* Readable typography
* No horizontal scrolling
* Consistent image cropping
* Responsive cards

Every page should be tested on:

* 360px
* 390px
* 414px
* 430px

---

# 13. Page Design Consistency

Every page should feel like part of the same website.

Consistency should be maintained across:

* Colours
* Typography
* Buttons
* Cards
* Icons
* Animations
* Image treatment
* Section spacing
* CTA design
* Footer

No page should introduce a completely different design language.

---

# 14. User Experience Principles

Every design decision should improve one or more of the following:

* Build trust with parents
* Inspire young footballers
* Showcase GCC's legacy
* Demonstrate professionalism
* Highlight academy achievements
* Encourage enquiries and registrations

The user should always understand what to do next.

---

# 15. Final Design Rules

Before approving any new page, confirm the following:

✓ Does it follow the GCC design language?

✓ Does it use the official colour palette?

✓ Does it use real academy photography?

✓ Does it maintain spacing consistency?

✓ Does it feel premium?

✓ Does it improve trust?

✓ Does it highlight football rather than decoration?

✓ Does it match the existing website?

If the answer to every question is "Yes", the design is approved.

---

# Final Design Statement

The GCC Chalissery Football Academy website is designed to present the academy as a premium, community-powered football institution built on legacy, discipline, and professional youth development.

Every future page, feature, and enhancement must strengthen this identity while maintaining the existing design language, ensuring a consistent and modern experience across the entire website.
