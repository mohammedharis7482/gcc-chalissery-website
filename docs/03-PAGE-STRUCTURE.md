# Purpose

This document defines the complete structure of every page on the GCC Football Academy website.

It serves as the implementation guide for development by specifying:

* The order of pages.
* The purpose of each page.
* The sections contained within each page.
* The content source for every section.
* The image usage strategy.
* The expected user journey.

The visual design must always follow **01-DESIGN-SYSTEM.md**, while all textual content and media references must follow **02-MASTER-CONTENT.md**.

---

# WEBSITE NAVIGATION

The primary navigation consists of the following pages:

1. Home
2. About
3. Academy
4. Our Team
5. Achievements
6. Gallery
7. Contact

The navigation should remain simple, clean, and consistent across desktop and mobile devices.

## Team Card Guidelines

* Use real photos only.
* Keep all profile cards visually consistent.
* Maintain equal card heights where possible.
* Prioritize clean spacing over large cards.
* Display image, name, role, and optional bio/qualifications.
* Highlight Management and Technical Directors slightly more through layout hierarchy, not oversized cards.

# IMAGE USAGE STRATEGY

All website images must come from:

`public/academy-content/`

Image selection should prioritize:

* High quality
* Clear football action
* Real academy moments
* Strong emotional impact

The media folder should be reviewed during development, and each image should be placed where it best supports the corresponding section. Images do not need to be manually renamed if they can be clearly identified during implementation.

---

# DEVELOPMENT NOTES

* Do not modify the approved design system.
* Preserve the existing layout and visual identity.
* Replace placeholder content with the finalized academy content.
* Use the best available real images throughout the website.
* Ensure all new sections match the existing design language.
* Maintain consistent spacing, typography, and animations across every page.
* Prioritize clean UI, readability, and visual hierarchy.
